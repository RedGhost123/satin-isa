Instruction bytes
Backward Compatibility: ISA v0.1 programs compile and run unchanged on Gen 3 hardware. Bytes [0..15] (upper half) are identical in layout to v0.1. Bytes [16..31] (lower half) are new — always zero in v0.1 programs. Use isa_upgrade_v1_to_v2() to convert a v0.1 binary to v2.0 format with safe defaults.
Endianness: Big-endian in memory. bit[255] = MSB of byte[0], bit[0] = LSB of byte[31]. All multi-byte fields stored big-endian: MSB at lower byte offset.
